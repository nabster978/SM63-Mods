# 2010 SM63 IGT Mod
Note: Only use this if you are performing Any% LDP or a category that requires the level designer to be unlocked at the beginning of the game.  Other wise use the normal IGT mod.

Timing starts automatically when the Start button is pressed on the main menu.

## Hotkeys
* , - Reset timer
* . - Stop timer
* ,+Enter - Reset timer, clear current file data and level designer clones, and return to title screen
* Shift+Tab - Change timer format
* Control+Tab - Change category

## Timer formats
* HH:MM:SS:mmm
	* H - Hours
	* M - Minutes
	* S - Seconds
	* m - Milliseconds
* HH:MM:SS:FF
	* H - Hours
	* M - Minutes
	* S - Seconds
	* F - Frames
* Frames
